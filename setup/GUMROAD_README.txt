JobMind Match Premium
=====================

INSTALL:
  1. Run JobMind-Match-Setup.exe
  2. Follow the setup wizard to the end
  3. JobMind Match opens automatically

NO PYTHON REQUIRED:
  Python is bundled inside the installer.
  You do NOT need to visit python.org.

DAILY USE:
  Start Menu or Desktop -> JobMind Match Premium

TO CLOSE:
  Start Menu -> Quit JobMind Match
