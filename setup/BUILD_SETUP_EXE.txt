Aap Gumroad par EK file bech sakte hain: JobMind-Match-Setup.exe
Buyer double-click kare → install → desktop shortcut → app chal jaye.


METHOD 1 — PROFESSIONAL setup.exe (RECOMMENDED)
────────────────────────────────────────────────

STEP A — Inno Setup install karein (free, one time)
  Download: https://jrsoftware.org/isdl.php
  Install "Inno Setup 6"

  NOTE: Agar "not found" aaye, yeh path check karein:
  %LOCALAPPDATA%\Programs\Inno Setup 6\ISCC.exe
  (User install — Program Files mein nahi hota)

STEP B — setup.exe banayein
  PowerShell / CMD:

    cd "D:\my apps\jobmind-match\installer"
    build-setup.bat

STEP C — Output file
  installer\output\JobMind-Match-Setup.exe

  Yeh file Gumroad par upload karein.


METHOD 2 — ZIP (agar setup.exe na bana payein)
──────────────────────────────────────────────
  cd installer
  prepare-staging.bat

  Phir folder "installer\staging" ko ZIP karein:
  JobMind-Match-Premium.zip

  Buyer extract karke START_HERE.bat chalaye.


KYA KARTA HAI setup.exe?
────────────────────────
  ✓ Files copy karta hai → C:\Users\...\AppData\...\JobMind Match
  ✓ Start Menu shortcut banata hai
  ✓ Desktop shortcut (optional checkbox)
  ✓ LICENSE + readme installer wizard mein dikhata hai
  ✓ Install ke baad app launch option
  ✓ Uninstall entry Windows Settings mein
  ✓ Owl icon installer par


BUYER KO PYTHON CHAHIYE?
────────────────────────
  Haan — setup.exe Python 3.11 check karta hai.
  Agar nahi hai to install python.org se (Add to PATH).

  Pehli run par START_HERE.bat:
    - venv banata hai
    - pip install karta hai
    - server start karta hai
    - browser khulta hai

TROUBLESHOOTING
───────────────
  "Inno Setup not found"
    → Install from jrsoftware.org, rerun build-setup.bat

  "Python not detected" on buyer PC
    → Buyer installs Python 3.11 with Add to PATH

  setup.exe blocked by Windows
    → Buyer: More info → Run anyway (unsigned exe normal for indie apps)

  Code signing (optional later)
    → Removes SmartScreen warning; costs ~$70-200/year cert
